# Writing Comments



### 1. 목적 및 용도
+ 이 프로젝트는 블로그, 게시판 등의 댓글 화면을 구현하기 위한 용도로 제작되었습니다.
+ SNS 로그인, 댓글 작성&수정&삭제, 댓글 투표, 금지어 기능, 도배 방지 기능을 담고 있습니다.
(단, sns 로그인은 더미데이터로 동작)



### 2. 사용 방법
+ 댓글 등록을 위해서 로그인 실행
+ 로그인 완료 후 댓글 등록
+ 댓글 등록 후 좋아요/싫어요 투표 가능
+ 작성한 댓글 수정/삭제 가능
+ 댓글에 금지어 포함 또는 도배할 경우 경고창
  + 금지어 : 비속어, 나쁜말, 바보
  + 도배 : 로그인 후 2번 이상 동일 문장 작성
